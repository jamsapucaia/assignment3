Assignment #3 - Item 2 - Topics

Create a page to set up an ordered list to show all the topics of this course (up to the chapter of using forms), which can be the same as the chapters listed in the course website. Then within each listed item use an unordered list to briefly describe at least 3 concepts, key points of knowledge and etc., that you think are the most important in that topic.

  1. Introduction to HTML
    • HTML stands for Hypertext Markup Language
    • HTML is a means of telling a web browser how to display a page
    • World Wide Web is a network of information resources located around the World
  2. Introduction to CSS
    • CSS stands for Cascading Style Sheets
    • Is used as a design tool
    • There are 3 types of Style Sheets:
      • Inline
      • Internal
      • External
  3. Working with text
    • Avoid using underlining "_", underlining is used mostly to indicate Links!
    • Sans-Serif typefaces are easir to read than Serif typefaces.
    • You can define the size of a font using different units:
      • Points (pt)
      • Percent (%)
      • Ems (em)
      • Pixels (px)
  4. Page Structure
    • Section is usually a chapter, but can also be used for general information of a Website.
    • Header usually hold the Title, Subtitle.
    • Footer can be used to store Copyright information, contact, and links.
  5. Working with links
    • There are two basic link types: Absolute and Relative.
    • Absolute links are those that include the entire URL:
      • https://www.google.com
    • Relative link do not use the entire URL of the referenced page. The pathname you use is relative to the current page.
      • Back to top.
  6. Working with Images
    • Common Web Image Formats are: APNG, AVIF, GIF, JPG, PNG, SVG, and WebP
      • Image file type and format guide | MDN
    • The standard file resolution for web graphics os 72 ppi.
    • Images can the linked from the web - Car image or stored locally like this Gif Image
  7. Creating Lists
    • This page is a good example of lists inside HTML code.
    • Lists can be Ordered (ol) 1, 2, 3, a, b, c, ...
      1. One
      2. Two
      3. Three
      4. Four
    • Lists can also be Unordered (ul):
      • Blue
      • Car
      • Apple
      • Shark
  8. Using Tables
    • Tables can be very helpful when you need to display tabular information.
    • Some exemples of tables are:
      • Sport scores.
      • To-do lists.
      • Shopping list.
      • Invoices.
    • Click here to see a Table showing Race Results.
  9. Emplying Forms
    • An HTML form is used to collect user input. The user input is most often sent to a server for processing.
    • Text input, radio buttons, check boxes, date, time, locations are some of the options for User Input.
    • It is very common to see a form to collect feedback or comments in a webpage.
